# coding-test

Java 코딩테스트 기초 학습 저장소

## 목표

백준과 프로그래머스를 활용해 Java 코딩테스트 기초 체력을 쌓는다.

## 학습 범위

* 입출력
* 사칙연산
* 조건문
* 반복문
* 배열
* 문자열
* 정렬
* 완전탐색
* 자료구조 기초

## 진행 방식

1. 문제를 푼다.
2. IntelliJ에서 실행 결과를 확인한다.
3. 정답 흐름을 기록한다.
4. GitHub에 커밋한다.

* CT03.java

문제 번호 순서대로 파일을 만든다.
